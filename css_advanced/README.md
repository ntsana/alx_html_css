CSS Advanced Project 

Welcome to the CSS Advanced Project! This project focuses on bringing a static HTML page to life using CSS. 
It’s all about styling and making the design visually appealing while staying true to the provided Figma reference.

Project Overview
The goal of this project is to practice advanced CSS techniques and replicate the design details provided in the Figma file. You’ll work on:

Structuring the HTML page using semantic tags.
Styling the page to match the design specifications.
Ensuring responsiveness for various screen sizes.

How to Get Started
Clone this repository to your local machine

Access the Design:
The Figma design is your reference. You’ll need to duplicate the file to your drafts.
Open this Figma link (or the one provided).
Click on “Duplicate to Drafts” to start working on it.

Install Fonts:
The project uses two custom fonts. Make sure to download and install them:
Source Sans Pro
Spin Cycle OT

Download Images:
Download all required images for the project here. Place them in an images folder inside the css_advanced directory.
